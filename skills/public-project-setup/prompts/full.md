# Полная подготовка проекта

Когда проект серьёзный и планируется рост сообщества.

## Полный чеклист

### Основа
- [ ] README.md с описанием, установкой, примерами, тестами
- [ ] LICENSE (MIT)
- [ ] .gitignore
- [ ] AUTHOR.md (контакт автора)

### Установка
- [ ] install.sh (Linux/macOS)
- [ ] install.ps1 (Windows PowerShell)

### Контрибьюторы
- [ ] .github/CONTRIBUTING.md
- [ ] .github/ISSUE_TEMPLATE/feature.md
- [ ] .github/ISSUE_TEMPLATE/bug-report.md
- [ ] .github/ISSUE_TEMPLATE/documentation.md

### Доказательства
- [ ] test-results/before/ — скриншоты без конфига
- [ ] test-results/after/ — скриншоты с конфигом
- [ ] Сравнительная таблица метрик

### GitHub
- [ ] Создать репозиторий
- [ ] Запушить всё
- [ ] Открыть начальные issues (фичи, баги, документация)
- [ ] Добавить topics/tags

## Алгоритм

```
1. Проверить/создать README.md
2. Создать install.sh + install.ps1
3. Создать .github/ с templates
4. Скопировать test-results
5. Обновить README секциями
6. Запушить
7. Открыть issues
```

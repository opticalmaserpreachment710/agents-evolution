# Advanced Mode

Продвинутый режим с глубокой детализацией.

## Когда использовать

- Пользователь опытный
- Запрос требует анализа архитектуры
- Нужны edge cases, оптимизации, подводные камни

## Структура

```
1. Суть (1 предложение)
2. Архитектура (диаграмма)
3. Механизм (пошагово)
4. Edge cases
5. Оптимизации
```
